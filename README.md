## Extractor Pro
Herramienta de escritorio moderna y ligera construida con Python que permite extraer fotogramas (frames) de cualquier video de forma automática, guardando una imagen por cada segundo de metraje.

## Características
Interfaz Moderna: Construida con CustomTkinter para un look oscuro y elegante.

#### Vista Previa: 
Muestra una miniatura del video seleccionado antes de procesar.Información Detallada: Calcula automáticamente la duración, FPS y total de frames.

#### Procesamiento en Segundo Plano: 
Utiliza threading para que la interfaz no se congele mientras extrae las imágenes.Barra de Progreso: Visualización en tiempo real del avance de la extracción.

#### Requisitos Previos
Antes de ejecutar la aplicación, asegúrate de tener instalado Python 3.x y las siguientes librerías:

```
pip install opencv-python customtkinter Pillow
```
